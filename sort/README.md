# sort
排序算法

![](https://github.com/BIGWangYuDong/sort/blob/main/sort/sort.png)

### 相关概念

**时间复杂度：**对排序数据的总操作次数，反应当n变化时，操作次数呈现的规律。

**空间复杂度：**算法在执行使所需要的存储空间度量。

**稳定：**如果a原本在b前面，而a=b，排序之后a仍然在b的前面。

**不稳定：**如果a原本在b的前面，而a=b，排序之后 a 可能会出现在 b 的后面。
